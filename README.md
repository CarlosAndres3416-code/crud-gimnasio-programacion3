# 💪 Sistema de Gestión de Inscripciones - Gimnasio (Equipo 2)

Proyecto final desarrollado para la materia utilizando el stack tecnológico solicitado. El sistema permite llevar el control total (CRUD) de los atletas, sus planes de entrenamiento, montos abonados y el estado de sus membresías.

## 🛠️ Tecnologías Utilizadas
* **Backend:** Node.js con Express
* **Base de Datos:** SQLite (librerías `sqlite3` y `sqlite`)
* **Frontend:** HTML5, CSS3 y JavaScript moderno (Fetch API)
